列表选项默认可见，方便用户进行比较选择，来选取单个选项

引入

import {RadioModule} from '~/cherry-ui;

基础用法

普通状态和禁用状态
<label cherry-radio [(ngModel)]="singleValue"> <span>独立使用 </span> </label> <label cherry-radio [(ngModel)]="singleValueDisabled" [uiDisabled]="true"> <span>独立使用</span> </label>

单选框组

使用一组单选框，group绑定当前选中的项
<cherry-radio-group [(ngModel)]="radioValueDisabled"> <label cherry-radio [uiValue]="'A'" [uiDisabled]="true"> <span>A</span> </label> <label cherry-radio [uiValue]="'B'" [uiDisabled]="true"> <span>B</span> </label> <label cherry-radio [uiValue]="'C'"> <span>C</span> </label> <label cherry-radio [uiValue]="'D'"> <span>D</span> </label> </cherry-radio-group> radioValueDisabled = 'A';

按钮型单选框

单选框的按钮样式
<cherry-radio-group [(ngModel)]="buttonValue"> <label cherry-radio-button [uiValue]="'A'"> <span>A</span> </label> <label cherry-radio-button [uiValue]="'B'" [uiDisabled]="true"> <span>B</span> </label> <label cherry-radio-button [uiValue]="'C'" [uiDisabled]="true"> <span>C</span> </label> <label cherry-radio-button [uiValue]="'D'"> <span>D</span> </label> </cherry-radio-group>

按钮型单选框尺寸

使用uiSize来使用不同尺寸大小的按钮型单选框
<cherry-radio-group [(ngModel)]="buttonSizeValue" [uiSize]="'sm'"> <label cherry-radio-button [uiValue]="'A'"> <span>A</span> </label> ... </cherry-radio-group> <cherry-radio-group [(ngModel)]="buttonSizeValue" [uiSize]="'lg'"> <label cherry-radio-button [uiValue]="'A'"> <span>A</span> </label> ... </cherry-radio-group>

API

cherry-radio

参数 说明 类型 默认值
ngModel 指定当前是否选中，支持双向绑定 Boolean -
uiDisabled 是否禁用单选框 Boolean false
uiValue 单选框的值 String -

cherry-radio-button(必须配合cherry-radio-group使用）

参数 说明 类型 默认值
uiDisabled 是否禁用单选框 Boolean false
uiValue 单选框的值 String -

cherry-checkbox-group

参数 说明 类型 默认值
ngModel 当前选中的值，支持双向绑定 String -
uiSize 尺寸大小，支队按钮型单选框生效 'sm'|'lg' -
