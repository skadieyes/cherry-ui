分页导航

引入

import {PaginationModule} from '~/cherry-ui;

基础用法

确定每页显示行数，和数据总数，分页器会进行分页
<cherry-paginator [rows]="10" [totalRecords]="100" [displayDirection]="'left'"></cherry-paginator>

显示页数

通过pageLinkSize确定分页器当前显示多少页
<cherry-paginator [rows]="10" [totalRecords]="100" [displayDirection]="'left'" [pageLinkSize]="3"></cherry-paginator>

每页条目

改变每页显示的行数
<cherry-paginator [rows]="10" [totalRecords]="100" [displayDirection]="'left'" [rowsPerPageOptions]="[10,20,30]"></cherry-paginator>

位置

改变分页器的位置，默认位置局中，可设为left或right
<cherry-paginator [rows]="10" [totalRecords]="100" [displayDirection]="'right'"></cherry-paginator>

API

cherry-paginator

参数 说明 类型 默认值
rows 每页行数 number 0
totalRecords 数据总量 number 0
displayDirection 显示位置 'left'|'right' -
rowsPerPageOptions 改变每页显示行数 array -
pageLinkSize 分页器当前显示页数 number 5
onPageChange 页数改变时的回调 function -
