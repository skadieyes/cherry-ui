cherryui是一个Angular组件库

使用cherryui来进行前端项目构建

1.下载

到版本库中下载cherryui

git clone ssh://username@10.253.46.121:29418/cherry-ui.git

2.依赖

在styles中或者cli中引入依赖的cherryui的css

@import '~cherry-ui/assets/sass/app'; @import '~cherry-ui/assets/cherryIcon/css/font.css';

组件中使用到了动画,引入动画构建模块

import {BrowserModule} from '@angular/platform-browser'; import {BrowserAnimationsModule} from '@angular/platform-browser/animations';

外部依赖,fontawesome字体图标库和material

"dependencies": { "font-awesome": "^4.7.0", "@angular/material": "^5.0.0" } "styles": [ "../node_modules/font-awesome/css/font-awesome.min.css", "../node_modules/@angular/material/prebuilt-themes/indigo-pink.css", ]

3.引入组件

可以在项目中引入组件了

import {CardModule} from '~/cherry-ui;
