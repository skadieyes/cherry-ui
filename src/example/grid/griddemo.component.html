grid模块用于网格的划分，将一行划分为24格, grid提供指令让你可以对网格的间距, 偏移, 排序， 对齐方式等属性进行设置。 grid同时还提供了响应式功能, 可以配合响应式指令来使用网格基础属性, 实现网格的响应式变化。

引入

import {GridModule} from '~/cherry-ui;

基础用法

grid将一行化为24列, 使用列来对你的元素进行布局。

网格顺序

通过push和pull改变网格的排列顺序,即改变网格的left、right

我一开始在左边 col-18 push-6
我一开始在右边 col-6 pull-18
<div cherry-row> <div cherry-col [uiSpan]="18" [uiPush]="6"> </div> <div cherry-col [uiSpan]="6" [uiPull]="18"> </div> </div>

API

cherry-row

参数 说明 类型 默认值
uiGutter 间隔 Number 0
uiType 布局模式,可设为flex String
uiAlign 垂直排列方式 String top
uiJustify 水平排列方式 String start

cherry-col

参数 说明 类型 默认值
uiSpan 网格占位格数，为 0 时相当于display:none Number -
uiOrder 网格顺序,flex模式下有效 Number 0
uiOffset 网格向右偏移数 Number 0
uiPush 网格向右移动数 Number 0
uiPull 网格向左移动数 Number 0
uiXs 小于768px的响应式网格 Number|Object -
uiSm 大于或等于768px的响应式网格 Number|Object -
uiMd 大于或等于992px的响应式网格 Number|Object -
uiLg 大于或等于1200px的响应式网格 Number|Object -
uiXl 大于或等于1600px的响应式网格 Number|Object -
