API

cherry-table

参数 说明 类型 默认值
value 传入表格的数据 array -
selectionMode 选择单选或多选 'single'|'multiple' -
selection 选中项 支持双向绑定 any -
headerTemplate 表格头模版 TemplateRef -
bodyTemplate 表格内容模版 TemplateRef -
captionTemplate 表格标题模版 TemplateRef -
footerTemplate 表格底部模版 TemplateRef -
paginator 是否使用分页器 boolean false
rows 每页行数 number 0
pagiDisplayDirection 分页器位置 'left'|'right' -
rowsPerPageOptions 改变每页显示行数 array -
columns 表格头数据 array -
onRowSelect 选择行事件回调 function -
onRowUnselect 取消选择行事件回调 fuction -
loading 加载状态 boolean false
scrollHeight 滚动高度 string -
scrollable 是否启用垂直滚动 boolean false
frozenColumns 锁定列数据 array -
frozenValue 锁定行数据 array -
frozenWidth 锁定列宽度 string -
reorderableColumns 启用拖拽排序 boolean false
responsive 响应式 boolean false
