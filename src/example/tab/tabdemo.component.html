标签组件拥有选项卡切换的功能

引入

import {TabModule} from '~/cherry-ui;

基础用法

tab的标准用法，默认线性tab

{{tab.name}} {{tab.content}}
<cherry-tabset> <cherry-tab *ngFor="let tab of tabs" [uiDisabled]="tab.disabled"> <ng-template #uiTabHeading> {tab.name} </ng-template> <span>{tab.content}</span> </cherry-tab> </cherry-tabset> tabs = [ { name: 'Tab 1', content: '这是Tab1的内容' }, { name: 'Tab 2', content: '这是Tab2的内容' }, { name: 'Tab 3', content: '这是Tab3的内容' }, { name: 'Tab 4', content: '禁用', } ];

图标

可以在tab标题中加入图标

{{tab.name}} {{tab.content}}
<cherry-tabset> <cherry-tab *ngFor="let tab of tabs" [uiDisabled]="tab.disabled"> <ng-template #uiTabHeading> <i [ngClass]="tab.icon"></i> {tab.name} </ng-template> <span>{tab.content}</span> </cherry-tab> </cherry-tabset> tabs = [ { name: 'Tab 1', content: '这是Tab1的内容'， icon: 'fa fa-lemon-o' } ... ];

位置

通过uiTabPosition来设置tab的排列位置，左右位置则变为垂直排列

{{tab.name}} {{tab.content}}
<cherry-select style="width: 120px" [(ngModel)]="position" [uiPlaceHolder]="'choose position'" > <cherry-option *ngFor="let option of options" [uiLabel]="option.label" [uiValue]="option" [uiDisabled]="option.disabled"> </cherry-option> </cherry-select> <cherry-tabset [uiTabPosition]="position.value"> <cherry-tab *ngFor="let tab of tabs" "> <ng-template #uiTabHeading> {tab.name} </ng-template> <span>{tab.content}</span> </cherry-tab> </cherry-tabset> options = [ { value: 'top', label: '上' }, { value: 'left', label: '左' }, { value: 'right', label: '右' }, { value: 'bottom', label: '下' } ]; position = this.options[0]; tabs = [ { name: 'Tab 1', content: '这是Tab1的内容' } ... ];

滑动

通过uiShowPagination启用滑动开关，当tab项较多在容器内不能完全呈现时，可以通过滑动来查看其他tab

{{tab.name}} {{tab.content}}
<cherry-select style="width: 120px" [(ngModel)]="position" [uiPlaceHolder]="'choose position'" > <cherry-option *ngFor="let option of options" [uiLabel]="option.label" [uiValue]="option" [uiDisabled]="option.disabled"> </cherry-option> </cherry-select> <cherry-tabset [uiTabPosition]="position.value" [uiShowPagination]="true" style="height:200px"> <cherry-tab *ngFor="let tab of tabs" "> <ng-template #uiTabHeading> {tab.name} </ng-template> <span>{tab.content}</span> </cherry-tab> </cherry-tabset> options = [ { value: 'top', label: '上' }, { value: 'left', label: '左' }, { value: 'right', label: '右' }, { value: 'bottom', label: '下' } ]; position = this.options[0]; tabs = [ { name: 'Tab 1', content: '这是Tab1的内容' } ... { name: 'Tab 17', content: '这是Tab17的内容' } ];

卡片风格

通过设置uiType为card 来使用卡片风格的tab

{{tab.name}} {{tab.content}}
<cherry-tabset [uiType]="'card'"> <cherry-tab *ngFor="let tab of tabs" "> <ng-template #uiTabHeading> {tab.name} </ng-template> <span>{tab.content}</span> </cherry-tab> </cherry-tabset> tabs = [ { name: 'Tab 1', content: '这是Tab1的内容' } ... ];

API

cherry-tabset

参数 说明 类型 默认值
uiType tab的两种样式，线形和卡片式 'line'|'card' -
uiTabPosition tab位置,可选top,right,bottom,left String 'top'
uiShowPagination 是否显示滑动按钮 Boolean false
uiSelectedIndex 当前激活的Tab Index，支持双向绑定 Number 0
uiSelectedIndexChange 当前激活的Tab Index回调 Func -
uiSelectChange 当前激活的Tab与Index值回调 Func -

cherry-tab

参数 说明 类型 默认值
#uiTabHeading tab的标题模版 ng-template -
uiDisabled tab禁用 Boolean false
uiClick 点击事件回调 Fun -
uiSelect 被选中后事件回调 Fun -
uiDeselect 不被选中事件回调 Fun -
