作为内容的载体,容器组件,可以与网格配合使用来进行页面布局

引入

import {CardModule} from '~/cherry-ui;

基础用法

内容
...
...
......
<cherry-card> 内容 </cherry-card>

区域划分

标题 操作区
内容
...
...
......
<cherry-card> <ng-template #title> 标题 </ng-template> 内容 <ng-template #operate> 操作区 </ng-template> </cherry-card>

收起展开

标题 操作区
内容
...
...
......
<cherry-card [toggleable]='true' > <ng-template #title> 标题 </ng-template> 内容 <ng-template #operate> 操作区 </ng-template> </cherry-card>

加载状态

标题 操作区
内容
...
...
......
<cherry-card [uiLoading]='true' > <ng-template #title> 标题 </ng-template> 内容 <ng-template #operate> 操作区 </ng-template> </cherry-card>

浮动强调

浮动强调默认开启,如不需要可设为false
内容
...
...
......
<cherry-card [uiHover]="true" > 内容 </cherry-card>

最大化卡片

通过设置uiMax来使你的卡片可以铺满全屏

标题 ... ... ... <cherry-card [[uiMax]="true" > 内容 </cherry-card>

API

参数 说明 类型 默认值
#title 卡片标题区域 ng-template -
#operate 卡片右上角的操作区域 ng-template -
toggleable 收起展开功能 Boolean false
uiHover 浮动强调 Boolean true
uiLoading 显示为加载状态 Boolean false
uiMax 卡片全屏最大化 Boolean false
