API

cherry-button

参数 说明 类型 默认值
uiType 可选择按钮的类型 String -
uiOutlineType 可选择轮廓按钮的类型 String -
uiSize 可设置按钮的大小 String default
uiShape 可以改变按钮形状为圆形 String -
uiLoading 显示加载状态 Boolean false

cherry-button-group

参数 说明 类型 默认值
uiSize 改变按钮组的大小 String default
uiVertical 改变按钮组的排列方式为垂直排列 Boolean false
