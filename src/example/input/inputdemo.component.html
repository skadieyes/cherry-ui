这部分包括文本输入框和文本域输入框,为用户提供输入的功能。我们的输入组件考虑了不同的输入情境,为不同的情境提供支持,并且使用这个输入组件你依然可以访问到底层的input标签。

引入

import {InputModule} from '~/cherry-ui;

基础用法

在input标签上添加cherry-input指令即可
<input cherry-input placeholder="正常状态"> <input cherry-input placeholder="禁用状态" [disabled]="true"> <input cherry-input placeholder="只读状态" [readonly]="true">

尺寸大小

使用uiSize来使用尺寸不同的输入框
<input cherry-input placeholder="sm" [uiSize]="'sm'"> <input cherry-input placeholder="normal"> <input cherry-input placeholder="lg" [uiSize]="'lg'">

文本域

使用uiAutosize来让文本域可以自动变化高度 可以设置uiAotosize的最小行数和最大行数
<textarea cherry-input placeholder="我是一个普通的文本域" rows="3" cols="3"></textarea> <textarea cherry-input placeholder="我是一个自动变化高度的文本域" [uiAutosize]="true"></textarea> <textarea cherry-input placeholder="我是最小行数为4，最大行数为6的自动变化高度的文本域" [uiAutosize]="{minRows: 4, maxRows: 6}"></textarea>

Group

为了满足更丰富的输入框展现形式和组合方式而存在的输入框容器,使用cherry-input-group标签来使用group容器

前置/后置标签组合

使用ng-template来运用前置后置标签

http://
.com
http:// .com
<cherry-input-group> <ng-template #addOnBefore>http://</ng-template> <input cherry-input placeholder="前置"> </cherry-input-group> <cherry-input-group> <input cherry-input placeholder="后置"> <ng-template #addOnAfter>.com</ng-template> </cherry-input-group> <cherry-input-group> <ng-template #addOnBefore>http://</ng-template> <input cherry-input placeholder="前置/后置"> <ng-template #addOnAfter>.com</ng-template> </cherry-input-group>

前缀/后缀标签组合

在输入框内加入前缀后缀标签
<cherry-input-group> <ng-template #prefix> <i class="fa fa-user"> </i> </ng-template> <input cherry-input placeholder="前缀"> </cherry-input-group> <cherry-input-group> <input cherry-input placeholder="后缀"> <ng-template #suffix> <i class="fa fa-chart"> </i> </ng-template> </cherry-input-group> <cherry-input-group> <ng-template #prefix> <i class="fa fa-user"> </i> </ng-template> <input cherry-input placeholder="前缀后缀"> <ng-template #suffix> <i class="fa fa-chart"> </i> </ng-template> </cherry-input-group>

输入框组合

使用uiCompact来把输入框进行紧密的组合
<cherry-input-group> <div cherry-col [uiSpan]="8"> <input cherry-input placeholder="name"> </div> <div cherry-col [uiSpan]="8"> <input cherry-input placeholder="name"> </div> <div cherry-col [uiSpan]="8"> <input cherry-input placeholder="name"> </div> </cherry-input-group> <cherry-input-group [uiCompact]="true"> <input cherry-input placeholder="name" style="width:30%"> <input cherry-input placeholder="name2" style="width:30%"> <input cherry-input placeholder="name3" style="width:30%"> </cherry-input-group> <cherry-input-group [uiCompact]="true"> <input cherry-input placeholder="起" style="width: 80px; text-align: center;"> <input cherry-input placeholder="~" style="width: 30px; border-left: 0px; pointer-events: none;"> <input cherry-input placeholder="止" style="width: 80px; text-align: center; border-left: 0px;"> </cherry-input-group>

输入框组合尺寸大小

使用uiSize来控制输入框组合的尺寸
<cherry-input-group [uiCompact]="true" [uiSize]="'sm'"> <input cherry-input placeholder="name" style="width:30%"> <input cherry-input placeholder="name2" style="width:30%"> <input cherry-input placeholder="name3" style="width:30%"> </cherry-input-group> <cherry-input-group [uiCompact]="true"> <input cherry-input placeholder="name" style="width:30%"> <input cherry-input placeholder="name2" style="width:30%"> <input cherry-input placeholder="name3" style="width:30%"> </cherry-input-group> <cherry-input-group [uiCompact]="true" [uiSize]="'lg'"> <input cherry-input placeholder="name" style="width:30%"> <input cherry-input placeholder="name2" style="width:30%"> <input cherry-input placeholder="name3" style="width:30%"> </cherry-input-group>

线型输入框

使用uiLine来运用一个视觉上更简洁的输入框

线型输入框中除了Input元素外还包括其他元素,所以需要group容器进行封装
<cherry-input-group [uiLine]="true"> <input cherry-input placeholder="normal"> </cherry-input-group> <cherry-input-group [uiLine]="true"> <cherry-input placeholder="禁用" [disabled]="true"> </cherry-input-group> <cherry-input-group [uiLine]="true"> <input cherry-input placeholder="只读" [readonly]="true"> </cherry-input-group> <cherry-input-group [uiLine]="true"> <ng-template #prefix> <i class="fa fa-user"> </i> </ng-template> <input cherry-input placeholder="前缀后缀"> <ng-template #suffix> <i class="fa fa-chart"> </i> </ng-template> </cherry-input-group>

输入框文本组合

提供垂直,flex两种文本展现方式,输入框获得焦点时,文本也能够和输入框一起获得焦点,uiLabel用来传入文本,uiLabelType可以选择展现方式

使用时把组件的父元素的position设为relative
<cherry-input-group [uiLabel]="'垂直'" [uiLabelType]="'row'"> <input cherry-input placeholder="row"> </cherry-input-group> <cherry-input-group [uiLine]="true" [uiLabel]="'垂直'" [uiLabelType]="'row'"> <input cherry-input placeholder="row"> </cherry-input-group> <cherry-input-group [uiLabel]="'flex'" [uiLabelType]="'flex'"> <input cherry-input placeholder="flex"> </cherry-input-group> <cherry-input-group [uiLine]="true" [uiLabel]="'flex'" [uiLabelType]="'flex'"> <input cherry-input placeholder="flex"> </cherry-input-group>

API

cherry-input

参数 说明 类型 默认值
uiSize 可设置输入框的大小 String -
uiAutosize textarea的特有Api boolean|object false

cherry-input-group

参数 说明 类型 默认值
#addOnBefore 设置前置标签 ng-template -
#addonAfter 设置后置标签 ng-template -
#prefix 设置前缀标签 ng-template -
#suffix 设置后缀标签 ng-template -
uiCompact 设置组合方式为紧密型 boolean false
uiSize 设置组合的大小 string -
uiLine 设置输入框为线型 boolean false
uiLabel 设置输入框文本标签 string -
uiLabelType 设置输入框文本标签类型 string -
