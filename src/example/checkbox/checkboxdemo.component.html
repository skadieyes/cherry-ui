一组默认可见的选项，可以进行多项选择。

引入

import {CheckboxModule} from '~/cherry-ui;

基础用法

普通状态和不可用状态
<label cherry-checkbox [(ngModel)]="_checked"> <span>Checkbox</span> <label> <label cherry-checkbox [uiDisabled]="true" [ngModel]="_checked"> <span>不可用</span> </label>

checkbox组

checkbox-group组件，可从数组生成一组checkbox
<cherry-checkbox-group [(ngModel)]="_checkgroups"> </cherry-checkbox-group>

垂直排列

将uiType设为vertical来让checkbox组垂直排列

<cherry-checkbox-group [(ngModel)]="_checkgroupsVertical" [uiType]="'vertical'"> </cherry-checkbox-group>

中间状态

使用uiIndeterminate来获得checkbox的中间状态（仅用于视觉反馈，如全选、半选的视觉区分）

<label cherry-checkbox [(ngModel)]="allChecked" (ngModelChange)="updateAllChecked()" [uiIndeterminate]="indeterminate"> <span>{allChecked ? 'All' : 'notAll'}</span> </label> <cherry-checkbox-group [(ngModel)]="_checkgroupsAll" (ngModelChange)="updateSingleChecked()"> </cherry-checkbox-group> allChecked = false; indeterminate = true; updateAllChecked() { this.indeterminate = false; if (this.allChecked) { this._checkgroupsAll.forEach(item => item.checked = true); }else { this._checkgroupsAll.forEach(item => item.checked = false); } } updateSingleChecked() { if (this._checkgroupsAll.every(item => item.checked === false)) { this.allChecked = false; this.indeterminate = false; } else if (this._checkgroupsAll.every(item => item.checked === true)) { this.allChecked = true; this.indeterminate = false; } else { this.indeterminate = true; } }

API

cherry-checkbox

参数 说明 类型 默认值
ngModel 指定当前是否选中，支持双向绑定 Boolean -
uiDisabled 是否禁用多选框 Boolean false
uiIndeterminate 半选状态 Boolean false

cherry-checkbox-group

参数 说明 类型 默认值
ngModel 选项数组，支持双向绑定 Array -
uiDisabled 是否禁用多选框 Boolean false
uiType 排列方式 'vertical' -
